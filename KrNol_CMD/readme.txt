I. Краткое описание приложения
    Приложение содзано для игры в "крестики-нолики" в режиме "пользователь против приоложения". Есть возможность выбора пользователем стороны, которая будет ходить первой. Есть возможность вызова руководства пользователя из приложения. Интерфейс – консольный. Управление осуществляется клавишами на клавиатуре.
II. Минимальные системные требования 
    Операционная система: Microsoft Windows 10 x64;
    Процессор: Процессор с частотой1 гигагерц (ГГц) или быстрее или система на кристалле SoC; 
    Видеоадаптер: DirectX 9 или более поздней версии с драйвером WDDM 1.0;  
    ОЗУ: 2 ГБ;
    Дисплей: 800x600;
    Интернет-соединение: необязательно.
III. Установка приложение
    Скопировать папку с приложением из архива в выбранную пользователем дирекотрию.
IV. Запуск приложения 
    Нажмите два раза по файлу main.exe.   
V. Работа с программой 

  I) Меню

     1. Для навигации в меню воспользуйтесь стрелкой "вверх" или стрелкой "вниз".
Для выбора нужного пункта, наведитесь на него с помощью стрелок и нажмите клавишу "enter"

     2. Пункт "Игра за крестики"
При выборе данного пункта игра запускается, при этом человек играет за крестики(ходит первым), компьютер играет за нолики(ходит вторым)

     3. Пункт "Игра за нолики"
При выборе данного пункта игра запускается, при этом человек играет за нолики(ходит вторым), компьютер играет за крестики(ходит первым)

     4. Пункт "Справка"
При выборе данного пункта открывается файл "readme.txt", в котором расписаны основные возможности программы 

     5. Пункт "Выход"
При выборе данного пункта происходит выход из программы 


  II) Игра 
     1. Управление
   Для навигации по полю используйте клавиши на клавиатуре:
      1) Для выбора ячейки, которая находится выше, выбранной в данной момент используйте "Стрелка вверх"
      2) Для выбора ячейки, которая находится ниже, выбранной в данной момент используйте "Стрелка вниз"
      3) Для выбора ячейки, которая находится левее, выбранной в данной момент используйте "Стрелка влево"
      4) Для выбора ячейки, которая находится правее, выбранной в данной момент используйте "Стрелка вправо"
   Если в выбранную сторону ячеек нет, то выбирается ячейка, противоположная той, что была выбрана до этого
   
   Для того, чтобы поставить крестик или нолик (в зависимости от выбранной сотроны) необходимо нажать клавишу "ENTER"
   Если ячейка уже занята, то программа сообщит пользователю об этой неприятной ситуации, ход будет пордолжаться до тех пор, пока крестик или нолик не будет поставлен в пустую ячейку (при условии если есть свободные ячейки, ситуация, при которой свободных ячеек не осталось рассматривается в дальнейшем)

   При нажатии неверной клавиши программа об этом заботливо сообщит.
   
    2. Победа, поражение и ничья

      Если одна из сторон ставит ТОЛЬКО крестики или ТОЛЬКО нолики в три ячейки по одной из горизонталей, одной из вертикалей, или одной из диагоналей, то она побеждает, противоположная сторона при этом проигрывает.
      При победе одной из сторон программа сообщает об этом и объявляет победившую сторону
      Если никто не создал выигрышную ситуацию, и при этом свободных ячеек на поле не осталось, то такая ситуация называется "Ничья", при этом никто не победил, но никто и не проиграл.
      При ничьей программа сообщает об этом интересном факте
      После вывода сообщения о выигрыше или ничьей, программа предлагает нажать любую клавишу, после чего происходит выход в меню
